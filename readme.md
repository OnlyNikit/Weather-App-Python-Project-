# 🌦️ Python Weather CLI App

A simple and professional command-line weather application built with Python. It fetches real-time weather information using the OpenWeather API and follows a clean, modular architecture.

## 🚀 Features

- 🌍 Search weather by city name
- 🌡️ Current temperature
- 🤗 Feels like temperature
- 💧 Humidity
- 🌬️ Wind speed
- ☁️ Weather description
- ⚠️ Proper error handling
- 🧩 Modular code structure
- ⏱️ Request timeout support

## 🛠️ Tech Stack

- Python 3
- Requests Library
- OpenWeather API

## 📂 Project Structure

```
weather-app/
```

## 📦 Installation

1. Clone the repository

```bash
git clone https://github.com/OnlyNikit/weather-app.git
```

2. Move into the project directory

```bash
cd weather-app
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

## 🔑 Get Your API Key

Create a free account on OpenWeather and generate an API key.

Replace:

```python
api_key = "YOUR_API_KEY"
```

with your own API key.

## ▶️ Run the Project

```bash
python main.py
```
